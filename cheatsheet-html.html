HTML Cheat Sheet

HTML Basic tags

  • <!DOCTYPE html> - Defines the document type and version of HTML
  • <html> - The root element of an HTML page
  • <Head> - Sets off the title and other info that isn't displayed
  • <Body> - Sets off the visible portion of the document
  • <title> - Puts name of the document in the title bar

HTML Lists

  • <ul> - Creates an unordered list
  • <ol start => ol> - Creates an ordered list (start= xx where xx is counting numbers)
  • <li> - Encompasses each list item
  • <dl> - Creates a definition list
  • <dl> - Precedes each definition list
  • <dt> - Precedes each definition term
  • <dd> - Precedes each definition

HTML Links

  • <a href="URL">clickable text /a> - Creates a hyperlink to a uniform resource locator
  • <a href="mailto:EMAIL_ADDRESS">clickable text /a> - Creates a hyperlink to an email address
  • <a name="NAME">> Creates a target location within documetn
  • <a href="#NAME">clickable text /a> - Creates a link to that target location

HTML Formatting

  • <p> - Creates a new paragraph
  • <br> - Inserts a line break
  • <blockquote>; - Puts content in a quote, indents text from both sides
  • <div> - Format block content with CSS
  • <span> - Format inline content with CSS
  • <b> - Turns text into bold
  • <strong> - tells the browser that the text is important
  • <i> - Turns text into italic
  • <mark> - Highlights the text
  • <u> This tag is used to underline text written between it
  • <sup> - displays the content slightly above the normal line
  • <sub> - It displays the content slightly below the normal line

HTML Text tags

  • <pre> - Creates preformatted text
  • <h1 => h6> - Creates headlines h1 = largest , h6 = smallest
  • <font size> - Sets size of the font 1 to 7 (use CSS instead)
  • <font color> - Sets font color (just use CSS)
  • <font face> - Defines the font used (just CSS please)
