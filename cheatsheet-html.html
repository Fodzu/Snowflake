HTML Cheat Sheet

  • <!DOCTYPE html> - Declaration for HTML5
  • <html> - Root element
  • <head> - Document metadata
  • <title> - Document title
  • <body> - Document body
  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (link)
  • <img> - Image
  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item
  • <div> - Division (container)
  • <span> - Inline container
